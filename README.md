# Our super duper project

- learn Git Basics
- Create a build pipeline with NPM scripts
- Create a small static website using the pipeline and git
- Kick the teacher out of the repository

## Installation instructions

- clone repository
- run `npm install`
- run `npm run build` to build the project for production
- run `npm run dev` to have a local server with live reload (will reload when CSS or JS sources changes)
